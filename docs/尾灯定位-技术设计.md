## 1. 模块定位

尾灯定位在车尾车辆内定位尾灯框，供特征提取。**用部件检测器替代旧 HSV 红色分割**（HSV 在灭灯/红色干扰下不准，已弃用，见 [原理.md](原理.md)）。

---

## 2. 方案：部件检测器定位尾灯

- 部件检测器（runs/detect/parts_v2）输出 taillight 类框
- 不依赖颜色（灭灯尾灯也定位）
- taillight mAP50=0.99、recall=0.967

## 4. 亮/灭灯判定

- 尾灯裁剪图转 HSV，取 V 通道（亮度）均值
- `V >= TAILLIGHT_MIN_V(150)` = 亮灯
- 不依赖 HSV 红色 mask，只取亮度（灭灯尾灯外壳暗）

> 阈值 150 经验值，适合夜间场景；白天灭灯尾灯 V 可能偏高，可按需调。

---

## 6. 集成

[detect_vehicle.py](../detect_vehicle.py) `detect_tail_light` 主流程：
```
车辆检测(detect_vehicle_full)
  -> 对每辆车: detect_taillights 找尾灯框
  -> 每个尾灯: crop_and_normalize + _judge_lit
  -> 可视化(红=亮灯/绿=灭灯) + 保存
```

识别程序（[recognize.py](../recognize.py)）在 rear 路径调用本模块定位尾灯。

## 7. 风险与对策

| 风险 | 对策 |
|---|---|
| 部件检测器漏检（buick/cadillac 58/62、43/45 rear 无尾灯） | [check_taillight_detection.py](../check_taillight_detection.py) 检查；补标注 + `build_all.py --skip-orientation` 重训部件 |
| 亮灭判定白天不准（灭灯尾灯 V 高） | 阈值 150 经验值；夜间为主 |
| 侧后方单尾灯可见 | 部件检测器检测可见尾灯即可 |
